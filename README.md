# PBO_Myfungsi_24079
PROJEK PBO UNTUK MEMENUHI MATA KULIAH PBO
Proyek Kendaraan merupakan sebuah aplikasi berbasis Python yang dibuat untuk mensimulasikan proses produksi kendaraan pada sebuah pabrik. Aplikasi ini memungkinkan pengguna memasukkan data kendaraan, memproses produksi, serta menampilkan daftar kendaraan yang telah diproduksi beserta total biaya produksi.

2. Tujuan Proyek

Tujuan pembuatan proyek ini adalah:

Menerapkan konsep Object Oriented Programming (OOP)

Melatih penggunaan class dan object dalam Python

Memahami proses produksi kendaraan secara sederhana

Membuat aplikasi interaktif menggunakan GUI (Tkinter)

3. Konsep Pemrograman yang Digunakan

Class dan Object

Inheritance & Enkapsulasi

Method dan Constructor

Exception Handling

GUI Tkinter

4. Struktur Class
a. Class Vehicle

Digunakan untuk merepresentasikan sebuah kendaraan dengan atribut:

Merek kendaraan

Model kendaraan

Warna kendaraan

Jenis mesin

Biaya produksi

Status produksi

Class ini juga memiliki method untuk mengatur status kendaraan dan menampilkan informasi kendaraan.

b. Class Factory

Berfungsi sebagai pengelola produksi kendaraan, yang bertugas:

Membuat objek kendaraan

Menyimpan daftar kendaraan

Menghitung total biaya produksi

5. Antarmuka Aplikasi (GUI)

Aplikasi menggunakan Tkinter sebagai antarmuka pengguna, yang terdiri dari:

Form input data kendaraan

Tombol produksi kendaraan

Tombol melihat daftar kendaraan

Area teks untuk menampilkan hasil

6. Cara Kerja Aplikasi

Pengguna memasukkan data kendaraan

Sistem memproses produksi kendaraan

Data kendaraan disimpan ke dalam daftar

Status kendaraan diperbarui

Informasi kendaraan dan total biaya ditampilkan

7. Manfaat Proyek

Membantu memahami konsep OOP dalam Python

Melatih pembuatan aplikasi desktop sederhana

Menjadi contoh simulasi sistem produksi

Cocok untuk tugas sekolah dan latihan pemrograman

8. Kesimpulan

Proyek Kendaraan ini menunjukkan bahwa Python dapat digunakan untuk membangun aplikasi desktop sederhana yang terstruktur dan interaktif. Dengan penerapan OOP dan GUI, program menjadi lebih rapi, mudah dikembangkan, dan mudah digunakan
